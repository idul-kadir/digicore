Untuk notifikasi whatsapp itu pake nomor sendiri atau sudah disediakan?

Untuk nomor whatsapp ada 2 pilihan, mau yang disediakan sendiri atau pake nomor random yang disediakan oleh DigiCore.

Jika menggunakan nomor sendiri jadi seperti apa skemanya? apakah nomornya kita kirim ke alamat DigiCore?

Tidak seperti itu kak. Jika menggunakan nomor sendiri, itu dipasang diperangkat kakak dan kita menyediakan qrcode untuk di scan.

Jika sambungan ke whatsapp terputus, bagaimana saya bisa tau?

Kami akan memberikan notifikasi ke whatsapp kakak jika sambungan terputus, dan bisa dilakukan scan kembali

Seperti apa itu VPN Tunnel?

VPN Tunnel yang kami tawarkan memungkinkan jaringan anda untuk di-online-kan, sehingga dapat diakses dari mana saja dengan aman melalui internet. Solusi ini sangat berguna bagi anda yang membangun webserver sendiri namun tidak memiliki IP Publik.

Apakah VPN Tunnel hanya berlaku untuk webserver?

Tidak. VPN Tunnel berlaku untuk semua perangkat agar bisa diremote dari jaringan internet. Baik itu router, mikrotik, alat IoT dan lain sebagainya

Trus, apa bedanya VPN Tunnel dengan IP Public?

Begini, untuk VPN Tunnel kita membuat 1 koneksi baik dengan SSTP, L2TP, Openvpn atau Wireguard. Setelah koneksi dibuat, maka user hanya bisa menggunakan 1 port. Jika ingin menambah penggunaan port, maka user harus membeli lagi VPN Tunnel-nya. Sedangkan jika menyewa IP Public, maka user bisa menggunakan port secara bebas tanpa ada biaya tambahan.

Kapan kita memutuskan untuk menyewa IP Public dan kapan kita menyewa VPN Tunnel?

Jika user hanya membutuhkan beberapa port untuk digunakan, sebaiknya memilih VPN Tunnel saja. Mengingat VPN Tunnel JAUH LEBIH MURAH daripada menyewa 1 IP Public

Apakah user bisa menggunakan lebih dari 1 layanan dalam 1 akun?

Bisa, jadi DigiCore menggunakan sistem saldo. Saldo anda akan dipotong sesuai dengan layanan yang anda gunakan.
